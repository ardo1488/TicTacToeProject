# TicTacToeProject

Link to Github:
    https://github.com/ardo1488/TicTacToeProject


How to run: 
    1). clone repository.
    2). open terminal or VS Code and navigate to the project folder.

    3). run the following commands:
        - npm install
        - npm start


    4). open browser and navigate to local server
